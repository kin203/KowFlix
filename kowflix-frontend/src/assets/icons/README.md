# KowFlix Icons

Bộ icon SVG được thiết kế riêng cho KowFlix.

## Style Guide
- **Size**: Default viewBox 24x24 (40x40 cho App Icon).
- **Stroke**: 2px, Round Cap, Round Join.
- **Color**: `currentColor` (Thừa hưởng từ text color).

## Usage
Màu mặc định nên là `#b3b3b3` (Inactive) và `#ffffff` hoặc `#e50914` (Active).
Ví dụ Tailwind: `text-gray-400 hover:text-white`.
